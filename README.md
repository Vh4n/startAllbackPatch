Open the .bat file in a text editor, 
Replace the variable 'CLSID' with your startAll Back CLSID folder name:

1- On regedit, go to: Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\CLSID\

2- Find the folder with lowercase letters.

i.e.: set "CLSID={6566e6f4-11ec-dc01-46cf-809bfb8ef08}"
